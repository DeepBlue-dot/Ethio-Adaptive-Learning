# Introduction to Quadratic Equations

## Definition
A quadratic equation is a polynomial equation of the second degree. It has the general form:

$$ax^2 + bx + c = 0$$

where $a \neq 0$, and $a$, $b$, and $c$ are constants.

## Standard Form
The standard form of a quadratic equation is:
$$ax^2 + bx + c = 0$$

### Examples:
- $x^2 + 5x + 6 = 0$
- $2x^2 - 3x + 1 = 0$
- $x^2 - 9 = 0$

## Solutions Using the Quadratic Formula

The roots of a quadratic equation can be found using:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### The Discriminant
The expression under the square root, $b^2 - 4ac$, is called the **discriminant**.

- If $b^2 - 4ac > 0$: two distinct real roots
- If $b^2 - 4ac = 0$: one repeated real root
- If $b^2 - 4ac < 0$: two complex conjugate roots

## Methods for Solving Quadratic Equations

### 1. Factoring
If the quadratic can be factored:
$$ax^2 + bx + c = (px + q)(rx + s) = 0$$

### 2. Completing the Square
Transform $ax^2 + bx + c = 0$ by manipulating it into the form $(x + p)^2 = q$

### 3. Using the Quadratic Formula
The most general method that works for all quadratic equations.

## Practice Problem
Solve: $x^2 + 5x + 6 = 0$

**Solution:**
Using the quadratic formula where $a = 1$, $b = 5$, $c = 6$:
$$x = \frac{-5 \pm \sqrt{25 - 24}}{2} = \frac{-5 \pm 1}{2}$$

Therefore: $x = -2$ or $x = -3$
