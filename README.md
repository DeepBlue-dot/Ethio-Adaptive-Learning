# Ethio-Adaptive-Learning 🇪🇹 🤖

An AI-powered, adaptive learning platform for the **Ethiopian Grade 12 National Examination (EHSLCE)**.

## 🚀 Overview
Addressing the performance gap in Ethiopia's national exams through **Intelligent Tutoring Systems (ITS)**. 

## 🧠 The "Brain"
- **KST (Knowledge Space Theory):** Curriculum prerequisite mapping.
- **IRT (Item Response Theory):** Student ability & question difficulty calibration.
- **BKT (Bayesian Knowledge Tracing):** Real-time mastery probability tracking.

## 🛠 Tech Stack
- **Frontend:** React.js / Tailwind CSS
- **Backend:** Node.js or Python
- **AI:** LLM-based Socratic Tutoring
