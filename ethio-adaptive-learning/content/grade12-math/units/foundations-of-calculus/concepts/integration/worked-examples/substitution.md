# Integration by Substitution

## Problem
Evaluate $\int 2x e^{x^2} dx$.

## Solution
1.  **Identify substitution**: Let $u = x^2$.
2.  **Find differential**: $du = 2x dx$.
3.  **Substitute**: The integral becomes $\int e^u du$.
4.  **Integrate**: $\int e^u du = e^u + C$.
5.  **Back-substitute**: Replace $u$ with $x^2$.

The final answer is $e^{x^2} + C$.
