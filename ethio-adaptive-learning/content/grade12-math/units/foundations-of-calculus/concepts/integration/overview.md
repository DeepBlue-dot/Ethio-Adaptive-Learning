# Integration

Integration is the reverse of differentiation. It allows us to reconstruct functions from their rates of change.

## Key Rules:
*   **Power Rule**: $\\int x^n dx = \\frac{x^{n+1}}{n+1} + C$.
*   **Constant Rule**: $\\int k dx = kx + C$.
*   **Definite Integral**: $\\int_a^b f(x)dx = F(b) - F(a)$.
