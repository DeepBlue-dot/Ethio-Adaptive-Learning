# Applying the Power Rule

## Problem
Calculate the derivative of the following function:
$f(x) = 4x^3 + 2x^2 - x + 5$

## Solution
To find the derivative $f'(x)$, we apply the power rule to each term individually:

1.  For $4x^3$: $\\frac{d}{dx}(4x^3) = 4 \\times 3x^{3-1} = 12x^2$
2.  For $2x^2$: $\\frac{d}{dx}(2x^2) = 2 \\times 2x^{2-1} = 4x$
3.  For $-x$: $\\frac{d}{dx}(-x^1) = -1 \\times 1x^{1-1} = -1$
4.  For $5$: The derivative of any constant is $0$.

Combining these results, we get:
$f'(x) = 12x^2 + 4x - 1$
