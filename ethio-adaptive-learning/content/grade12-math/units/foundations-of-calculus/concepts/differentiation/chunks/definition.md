# Formal Definition of Derivative

The derivative of a function $f(x)$ at $x = a$ is defined as the limit:

$$f'(a) = \\lim_{h \\to 0} \\frac{f(a+h) - f(a)}{h}$$

If this limit exists, we say the function is differentiable at $a$. This value represents the slope of the tangent line to the graph of $f$ at the point $(a, f(a))$.
