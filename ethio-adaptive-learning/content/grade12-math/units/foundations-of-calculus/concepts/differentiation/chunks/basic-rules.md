# Basic Differentiation Rules

To differentiate functions efficiently, we use the following rules:

1.  **Constant Rule**: $\\frac{d}{dx} c = 0$
2.  **Power Rule**: $\\frac{d}{dx} x^n = nx^{n-1}$
3.  **Sum Rule**: $(f + g)' = f' + g'$
4.  **Constant Multiple Rule**: $(cf)' = c \\cdot f'$

Using these rules, we can differentiate any polynomial function easily.
