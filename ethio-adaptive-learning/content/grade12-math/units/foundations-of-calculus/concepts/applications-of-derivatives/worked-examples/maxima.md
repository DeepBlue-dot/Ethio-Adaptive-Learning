# Finding a Local Maximum

## Problem
Find the local maximum of $f(x) = -x^2 + 8x - 10$.

## Solution
1.  **Find the derivative**: $f'(x) = -2x + 8$.
2.  **Set to zero**: $-2x + 8 = 0 \\implies x = 4$.
3.  **Check nature**: $f''(x) = -2$. Since the second derivative is negative, it is a local maximum.
4.  **Calculate $y$**: $f(4) = -(16) + 32 - 10 = 6$.

The local maximum is at $(4, 6)$.
