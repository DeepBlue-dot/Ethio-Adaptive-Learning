For continuous functions such as polynomials, direct substitution is the first strategy:

1. identify the target value of `x`,
2. substitute it into the expression,
3. simplify.

For example, if `f(x) = x^2`, then:

`\\lim_{x \\to 2} x^2 = 2^2 = 4`

This works because `x^2` has no break or jump at `x = 2`.
