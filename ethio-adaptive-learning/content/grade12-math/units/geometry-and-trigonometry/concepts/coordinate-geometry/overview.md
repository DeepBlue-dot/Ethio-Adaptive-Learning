# Coordinate Geometry

Coordinate geometry (also known as analytic geometry) links algebra and geometry using a coordinate system.

## Key Formulas:
*   **Distance**: $d = \\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
*   **Midpoint**: $M = \\left(\\frac{x_1+x_2}{2}, \\frac{y_1+y_2}{2}\\right)$
*   **Circle**: $(x-h)^2 + (y-k)^2 = r^2$
