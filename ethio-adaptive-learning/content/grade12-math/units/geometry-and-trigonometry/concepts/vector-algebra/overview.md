# Vector Algebra

Vectors represent quantities with both size and direction. They are essential for physics and engineering.

## Key Operations:
*   **Magnitude**: $|\mathbf{v}| = \sqrt{a^2 + b^2}$.
*   **Addition**: $(a, b) + (c, d) = (a+c, b+d)$.
*   **Dot Product**: $\mathbf{u} \cdot \mathbf{v} = u_1v_1 + u_2v_2$.
