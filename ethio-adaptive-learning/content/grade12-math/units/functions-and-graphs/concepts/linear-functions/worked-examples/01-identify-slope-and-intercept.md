# Problem
For the function `f(x) = 2x + 3`, identify the slope and the `y`-intercept.

# Solution
Compare the equation to the template `f(x) = mx + b`.

- The coefficient of `x` is `2`, so the slope is `2`.
- The constant term is `3`, so the `y`-intercept is `3`.

This means the graph rises `2` units for every `1` unit moved to the right and crosses the `y`-axis at `3`.
