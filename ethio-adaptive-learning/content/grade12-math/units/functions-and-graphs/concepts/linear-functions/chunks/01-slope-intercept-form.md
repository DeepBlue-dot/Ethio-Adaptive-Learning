The standard teaching form for a linear function is:

`f(x) = mx + b`

Interpretation:

- `m` controls steepness and direction.
- `b` tells you where the graph meets the `y`-axis.

If `m > 0`, the line rises from left to right. If `m < 0`, the line falls from left to right.

When students are given an equation, they should be able to identify slope and intercept immediately before doing any substitution or graph sketching.
