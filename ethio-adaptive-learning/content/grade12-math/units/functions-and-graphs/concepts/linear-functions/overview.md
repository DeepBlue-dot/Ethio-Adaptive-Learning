Linear functions model relationships with a constant rate of change. In Grade 12 work, students should read a line from an equation, a table, or a graph and move between those representations with confidence.

Use the form `f(x) = mx + b` as the default reference point:

- `m` is the slope, or the amount the output changes for each unit change in `x`.
- `b` is the `y`-intercept, or the value of the function when `x = 0`.

This concept unlocks later graph and calculus topics, so the goal is fluency rather than one-off recall.
